# Flask Web Application

## Overview
A simple Flask web application with multiple pages (Homepage, Blog, Calculator).

## Project Structure
- `app.py` - Main Flask application
- `templates/` - HTML templates for each page
  - `homepage/` - Homepage templates
  - `blog/` - Blog templates
  - `calculator/` - Calculator templates
- `static/` - Static assets (CSS, JS, images)
  - `homepage/` - Homepage static files
  - `blog/` - Blog static files
- `requirements.txt` - Python dependencies

## Running the Application
The application runs on port 5000 using the Flask development server.

## Dependencies
- Flask
- Gunicorn (for production)
